### Software & DevOps

#### Ops-Runner

Modern web application built to help operators handle day-to-day operations.

Technology Stack: ReactJS, Go, and Redis

#### ChatOps for Microsoft Teams

Automated deployments triggered directly from Microsoft Teams as native chat queries.

- Developers initiate deployments as chat queries
- Automated, transparent deployment status updates
- Faster turnaround for product feature development

Technology Stack: NodeJS, Jenkins, and MS Teams

#### GoVuln Scanner [WIP]

A hobby project, built from the ground up in Go to mimic a CIS audit / vulnerability scanner.

- GUI scanner with remote execution
- Conformance with latest CIS benchmark standards (v2)
- Remote scanning over SSH

[Source Code](https://github.com/karthick-kk/govuln-scanner)

#### Route53 Updater for RDS

Python utility to create/update CNAME records of a hosted zone with the RDS cluster endpoint.

[Source Code](https://github.com/karthick-kk/aws-automation)

### Android ROM & Kernel Development

I build custom Android ROMs and kernels for Xiaomi devices, publishing builds and device trees publicly.

#### vili (Xiaomi 11T Pro)

Custom ROMs and kernels for the vili platform, maintained under the `kknixx-extended-roms` project.

- ROM builds (Evolution-X, PixelOS, Lineage, Rising): [SourceForge](https://sourceforge.net/projects/kknixx-extended-roms/files/vili/)
- Kernels (RED kernel): [SourceForge](https://sourceforge.net/projects/kknixx-extended-roms/files/vili/Kernel/)
- Device & kernel trees: [github.com/vili-develop](https://github.com/vili-develop)

#### dada (Xiaomi device)

Device trees and kernels for the dada platform.

- Device & kernel trees: [github.com/dada-devel](https://github.com/dada-devel)

#### Kernels

- **RED kernel** — performance-oriented kernel for vili (AOSP/MIUI), built and released via SourceForge.
- **SilverCore** — custom kernel project for supported Xiaomi devices.
